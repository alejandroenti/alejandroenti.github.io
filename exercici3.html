Enllaços externos amb atributs

Enllaços a diferents pàgines web:

  • Aquest enllaç té l'atribut target amb el valor _self. Això l'obre al mateix context on ens trobem ara. Aquest és el comportament per defecte - Wikipedia en castellà

  • Aquest enllaç té l'atribut target amb el valor _top. Això l'obre al context més alt de navegació (si no hi ha parent es comporta com _self) - Web del centre

  • Aquest enllaç té l'atribut target amb el valor _parent. Això l'obre al context de nevegació del pare (parent). Si no hi ha parent es comporta com _self - Youtube

  • Aquest enllaç té l'atribut target amb el valor _blank. Això l'obre a un nou context de navegació, usualment una pestanya nova - Chat GPT
